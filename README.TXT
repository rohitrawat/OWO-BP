Please run the program mlp_TRAIN_CLASS for training and validating a Neural
Network. Open MATLAB and change the directory to where this program is stored.
Then in the Current Folder (in MATLAB's desktop), right click on the program and 
select Run. Or if you are in the correct directory, you can type the name of the
program (mlp_TRAIN_CLASS) and press enter.

The program should prompt you for input upon running.
You should provide a training and validation file for it to run.

Please run mlp_PROCESSING_CLASS to obtain processing results (if needed).
